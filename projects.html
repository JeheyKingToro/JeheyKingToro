  • Creating a TicTacToe Game using Embedded Systems

    It was a group project consisting of three members. We work together to create a TicTacToe game using FRDM-K20D50M microcontroller, a LCD display, 4x4 Keypads and a buzzer. The buzzer is used to signal the winner of the game and button sounds. The LCD display is used to display the current state of the game board. The 4x4 keypad is used to input player moves. The game of TicTacToe is to played between two players where each player takes turns to place their symbol (X or O) on the game board. The winner is the player who can get three of their symbols in a row, column, or diagonal. Find here is the link to the project video demo:

  • Gearbox Design using SolidWorks

    It was a group project where we designed a functional gearbox using SolidWorks. The project involved creating detailed 3D models of the gearbox components, assembling them, and performing FEA simulations to analyze performance under various conditions. The main objective was to design the perfect gear ratio for the gearbox to ensure optimal performance to lift a 2KG load to a height of 1.5meter. This project showcased my proficiency in mechanical design, kinematics, and CAD software. Unfortunately, there is no link to the project report. But there is a video demo of the project:
